# Triple Espresso
Nombre del Proyecto: Página web de una cafetería. 
Descripcion del proyecto: Pagina de la cafeteria de Triple Espresso para la reserva de mesas. se utilizaron las siguientes tecnologia:
* HTML
* CSS
Planes de mejora del proyecto: Conectar la pagina a una base NoSQL que gestione las reservaciónes.
